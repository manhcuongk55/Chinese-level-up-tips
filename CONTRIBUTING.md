# Contributing to Chinese Level Up Tips

Thank you for considering contributing! 🎉

## How to Contribute

### 📝 Content Contributions
- Fix typos or improve explanations
- Add new vocabulary entries
- Improve example sentences
- Add resource recommendations

### 🌍 Translations
- Translate chapters into your language
- Create a `docs/<lang-code>/` folder
- Follow the existing structure

### 🐛 Issues
- Report errors or inaccuracies
- Suggest new chapter topics
- Share useful resources

## Submission Guidelines

1. Fork the repository
2. Create a feature branch: `git checkout -b improve/chapter-3`
3. Make your changes
4. Commit: `git commit -m "feat: improve pinyin explanation in chapter 3"`
5. Push and open a Pull Request

## Content Standards

- Write in **clear, accessible English**
- Include Pinyin for all Chinese text
- Include English translations
- Cite sources when referencing research
